
### Cloning the repository

```shell
git clone https://github.com/YevheniiaSimaka/Airbnb-Clone
```

### Install packages

```shell
npm i
```

### Setup .env file


```js
DATABASE_URL=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GITHUB_ID=
GITHUB_SECRET=
NEXTAUTH_SECRET=
```

### Setup Prisma

```shell
npx prisma db push

```

### Start the app

```shell
npm run build
npm start
```

### Start the app for development

```shell
npm run dev
```
